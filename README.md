# My CV

This CV is available live at [www.jaussoin.com](https://www.jaussoin.com)

Build the static site and the downloadable PDF with:

- `npm run build`

The build writes the website to `dist/` and generates a print-ready PDF at `dist/downloads/Antoine_Jaussoin_CV.pdf`.

The DOCX file remains in `public/downloads/Antoine_Jaussoin_CV.docx`.

How to print:

- No margin
- Scale 60%
