# My CV

This CV is available live at [www.jaussoin.com](https://www.jaussoin.com)

Generate the downloadable documents with:

- `make generate`

This writes the DOCX and PDF files to `public/downloads/` and refreshes the built PDF in `dist/downloads/Antoine_Jaussoin_CV.pdf`.

Build the static site with:

- `npm run build`

The build writes the website to `dist/` without generating the PDF inside the Docker image build.

How to print:

- No margin
- Scale 60%
